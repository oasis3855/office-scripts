## 概要

Excelワークブック内の全ワークシートから、コメントのあるセル一覧を表示する。

![コメントのあるセル一覧を表示するダイアログ](readme_pics/find-comment-dlg.jpg)


また、コメントのあるセルから、コメントを削除することもできる。

![コメントのあるセル一覧を表示し、新規保存するダイアログ](readme_pics/find-comment-dlg-savenew.jpg)
